    1.2 算法

    算法五个条件：输入、输出、明确性、有限性、有效性

    算法效能

    1. 空间复杂度
    2. 时间复杂度

    Big-oh：O(f(n))：某算法在计算机中运行时间不会超过某一常数倍数的f(n)

    $$
    证明：\sum_{(1\leqslant i\leqslant n)} i＝O(n^2) \ \sum_{(1\leqslant i\leqslant n)} i＝1+2+3+\cdots+n= \frac {n(n+1)}{2} \可以找到常数n_0、c=1,当n\geqslant n_0,\frac {n(n+1)}{2}\leqslant n^2,因此时间复杂度O(n^2)
    $$

    Big-oh 说明
    O(1) 常数时间
    O(n) 线性时间
    O(log₂n) 次线性时间
    O(n²) 平方时间
    O(n³) 立方时间
    O(2ⁿ) 指数时间
    O(nlog₂n) 线性乘对数时间

    $$
    n\geqslant16时：O(1)<O(log₂n)<O(n)<O(nlog₂n)<O(n²)<O(n³)<O(2ⁿ)
    $$

    Big-omega：Ω：运行时间度量最好情况

    Big-theta： θ：更精确，确定上限和下限

    常见算法：

    1. 分治法：将一个大问题分割成多个子问题。

    2. 递归法：一个函数或子程序由自身定义或调用。

      //斐波那契数列
public static int Fibonacci(int n){
    if(n==0) return 0;
    else if(n==1) return 1;
    else return (Fibonacci(n-1)+Fibonacci(n-2));
}

    3. 贪心法：从某一点开始采取当前状态下最有利或最优的选择，持续在每一步中选择最佳方法。

    4. 动态规划法：将大问题拆成子问题的同时储存子问题的答案

    5. 迭代法：无法用公式一次解决而需要使用迭代

    6. 枚举法：逐一列举问题的解答

    2. 数组结构

    2.1 线性表

    线性表：n个元素的有限序列

    数据存储结构：

    • 静态数据结构（密集表）：使用连续分配的内存空间储存有序表中的数据
      • 优点：设计简单，读取和修改任意元素时间固定
      • 缺点：删除或加入时间时需要移动大量数据
    • 动态数据结构（链表）：使用不连续的内存空间储存具有线性表特性的数据
      • 优点：插入或删除数据方便，节省内存
      • 缺点：设计麻烦，查找数据需要按顺序

    数组（Array）：一排紧密相邻的可数内存，并提供了能直接访问单一数据内容的计算方法

    4.1 模拟堆栈

    • 数组模拟堆栈：设计简单但是大小确定
    import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

//堆栈的类声明
class StackByArray{
    private int[] stack;//类中声明数组
    private int top;//指向堆栈顶端的索引
    //类构造函数
    public StackByArray(int stackSize){
        stack = new int[stackSize];
        top = -1;
    }
    //类方法：push 存储数据
    public boolean push(int data){
        if(top >= stack.length){//判断堆栈顶指针是否大于数组大小
            System.out.println("堆栈已满，无法放入数据");
            return false;
        }else{
            stack[++top]=data;
            return true;
        }
    }
    //类方法：empty 判断是否为空
    public boolean empty(){
        if(top==-1) return true;
        else return false;
    }
    //类方法：pop 弹出数据
    public int pop(){
        if(empty()) return -1;
        else return stack[top--];//数据弹出，指针下移
    }
}

public class ch_4_1_1 {
    public static void main(String[] args) throws IOException {
        //存储数据
        BufferedReader  buffer;
        int value;
        StackByArray stack = new StackByArray(10);
        buffer = new BufferedReader(new InputStreamReader(System.in));
        System.out.println("请输入10个数据");
        for(int i=0;i<10;i++){
            value = Integer.parseInt(buffer.readLine());
            stack.push(value);
        }
        //输出数据

            System.out.println("堆栈中弹出数据顺序："+stack.pop());
        }
    }
}
    • 链表模拟堆栈：可以动态改变链表长度，避免空间浪费
    import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

//声明链表节点
class Node{
    int data;
    Node next;
    public Node(int data){
        this.data = data;
        this.next = null;
    }
}
//构造类
class StackByLink{
    public Node front;//指向堆栈底部的指针
    public Node rear;//指向堆栈顶端的指针
    //类方法：isEmpty 若堆栈为空，front=null
    public boolean isEmpty(){
        return front==null;
    }
    //类方法：insert 将数据压入堆栈顶
    public void insert(int data){
        Node newNode = new Node(data);
        if(this.isEmpty()){
            front = newNode;
            rear = newNode;
        }else{
            rear.next = newNode;
            rear = newNode;
        }
    }
    //类方法：pop 弹出数据
    public void pop(){
        Node newNode;
        if(this.isEmpty()){
            System.out.println("当前为空堆栈");
            return;
        }
        newNode = front;
        if(newNode == rear){
            front = null;
            rear = null;
            System.out.println("当前为空堆栈");
        }else{
            while (newNode.next!=rear){
                newNode = newNode.next;
            }
            newNode.next = rear.next;
            rear = newNode;
        }
    }
    //类方法：printStack 输出堆栈中的数据
    public void printStack(){
        Node current = front;
        while (current!=null){
            System.out.print("["+current.data+"]");
            current = current.next;
        }
        System.out.println();
    }
}
public class ch4_1_3 {
    public static void main(String[] args) throws IOException {
        BufferedReader buffer = new BufferedReader(new InputStreamReader(System.in));
        StackByLink stack = new StackByLink();
        int choice = 0;
        while (true){
            System.out.println("0:结束，1：将数据存入，2：将数据弹出");
            choice = Integer.parseInt(buffer.readLine());
            if(choice == 0){break;}
            else if(choice == 1){
                System.out.println("请输入存储数据");
                choice = Integer.parseInt(buffer.readLine());
                stack.insert(choice);
                System.out.println("当前堆栈内容：");
                stack.printStack();
            }else if(choice == 2){
                stack.pop();
                System.out.println("当前堆栈内容：");
                stack.printStack();
            }else {
                System.out.println("输入错误");
            }
        }
    }
}

    7. 图结构

    图：由顶点和边组成的集合，用 G=(V,E) 表示

    • 无向图：同一条边两个顶点没有次序关系。(V1,V2)与(V2,V1)一样

    7.3 生成树

    生成树：以最少边来连通图中所有顶点，且不造成回路的树结构

    最小生成树：在树的边上加权重值

    Prim算法

    对一个加权图形 G=(V,E) ,设 V={1,2,3…n},U={1},V和U表示两类顶点集合。从U-V差集中找一个点x，若x与U中某点形成最小成本的边且不造成回路，则将x加入U，反复执行该操作直至U=V。

    Kruskal算法

    将各边按权值从小到大排列，从权值低的边开始建立最小生成树，若造成回路则舍弃，直至加入n-1个边

    public class ch7_6 {
    public static int VERTS = 6;
    public static int[] v = new int[VERTS+1];
    public static Node1 NewList = new Node1();
    public static int findmincost(){
        int minval = 100;
        int retptr = 0;
        int a= 0;
        while (NewList.next[a] != -1){
            if(NewList.val[a]<minval && NewList.find[a]==0){
                minval = NewList.val[a];
                retptr = a;
            }
            a++;
        }
        NewList.find[retptr] = 1;
        return retptr;
    }

    public static void mintree(){
        int result = 0;
        int mceprt;
        int a = 0;
        for (int i = 0; i < VERTS; i++) {
            v[i]=0;
        }
        while (NewList.next[a]!=-1){
            mceprt = findmincost();
            v[NewList.from[mceprt]]++;
            v[NewList.to[mceprt]]++;
            if(v[NewList.from[mceprt]]>1 && v[NewList.to[mceprt]]>1){
                v[NewList.from[mceprt]]--;
                v[NewList.to[mceprt]]--;
                result = 1;
            }else {
                result = 0;
            }
            if(result == 0){
                System.out.println("起始顶点["+NewList.from[mceprt]+"];终止顶点["+NewList.to[mceprt]+"];路径长度["+NewList.val[mceprt]+"]");
            }
            a++;
        }
    }

    public static void main(String[] args) {
        //声明图数组
        int Data[][] = {{1,2,6},{1,6,12},{1,5,10},{2,3,3},{2,4,5},{2,6,8},{3,4,7},{4,6,11},{4,5,9},{5,6,16}};
        int DataNum,fromNum,toNum,findNum,Header=0,FreeNode;
        System.out.println("建立图的链表：");
        for (int i = 0; i < 10; i++) {
            for (int j = 0; j < VERTS; j++) {
                if(Data[i][0] == j){
                    fromNum = Data[i][0];
                    toNum = Data[i][1];
                    DataNum = Data[i][2];
                    findNum = 0;
                    FreeNode = NewList.findFree();
                    NewList.create(Header,FreeNode,DataNum,fromNum,toNum,findNum);
                }
            }
        }
        NewList.printList(Header);
        System.out.println("建立最小生成树：");
        mintree();
    }
}
class Node1{
    int MaxLen = 20;//链表最大长度
    int from[] = new int[MaxLen];
    int to[] = new int[MaxLen];
    int find[] = new int[MaxLen];
    int val[] = new int[MaxLen];
    int next[] = new int[MaxLen];
    public Node1(){
        for (int i = 0; i < MaxLen; i++) {
            next[i] = -2;//-2 表示未用节点
        }
    }
    //搜索可用节点
    public int findFree(){
        int i;
        for ( i = 0; i < MaxLen; i++) {
            if(next[i] == -2) break;
        }
        return i;
    }
    //建立链表
    public void create(int Header,int FreeNode,int DataNum,int fromNum,int toNum,int findNum){
        int Pointer;//当前节点位置
        if(Header == FreeNode){
            val[Header] = DataNum;
            from[Header] = fromNum;
            to[Header] = toNum;
            find[Header] = findNum;
            next[Header] = -1;
        }else {
            Pointer = Header;
            val[FreeNode] = DataNum;
            from[FreeNode] = fromNum;
            to[FreeNode] = toNum;
            find[FreeNode] = findNum;
            next[FreeNode] = -1;
            while (next[Pointer] != -1){
                Pointer = next[Pointer];
            }
            next[Pointer] = FreeNode;
        }
    }
    //打印链表
    public void printList(int Header){
        int Pointer = Header;
        while (Pointer != -1){
            System.out.println("起始顶点["+from[Pointer]+"];终止顶点["+to[Pointer]+"];路径长度["+val[Pointer]+"]");
            Pointer = next[Pointer];
        }
    }
}

    1.冒泡排序（Bubb Sort） 稳定。空间复杂度最佳O(1)
    2.选择排序（Selection Sort） 不稳定。空间复杂度最佳O(1)
    3.插入排序（Insertion Sort） 稳定。空间复杂度最佳O(1)
    4.希尔排序（Shell Sort） 稳定。空间复杂度最佳O(1)
    1.快速排序（Quick Sort） 不稳定。最差O(n),最佳O(log₂n)
    2.堆积排序（Heap Sort） 不稳定。空间复杂度最佳O(1)
    3.基数排序（Radix Sort） 稳定。O(np)，n为数据个数，p为基底

    8.1 排序分类

    内部排序

    排序的数据量小，可以全部加载到内存中排序

    冒泡排序

    从第一个元素开始比较相邻元素大小，若大小有误对调顺序再进行下一个元素比较

    选择排序

    在所有数据中若从小到大排则将最大的放在第一个位置，依次排序。若从大到小则反之

    插入排序

    将数组中元素逐一与已排好的数据比较并插入

    希尔排序

    将数据区分成特定间隔的几个小区块，以插入排序法排完区块内的数据再减小间隔距离

    归并排序

    针对已排序好的两个或两个以上数列通过合并的方式组合成一个

    快速排序

    在数据中找到一个中间位置，将比其数据小的放在其左侧，大的放在右侧，再用同样的方式处理左右两边数据

    堆积排序

    选择排序的改进版，利用堆积树完成

    基数排序

    是一种分配排序方式

    • 最高位优先（MSD）：从左边位数开始比较

    • 最低位优先（LSD）：从右边位数开始比较

    外部排序

    排序数据量大，需通过硬盘等辅助储存器排序

    直接合并排序法

    将排序文件分成几个小文件用内部排序法排序，再将小文件两两合并最后合并成一个文件

    K路合并法

    即将直接合并法分成k个

    9. 查找

    9.1 查找方法

    顺序查找法：将数据一项一项按顺序逐个查找

    二分查找法：将数据分成两部分，将键值与中间值比较，由此可知数据所在位置

    插值查找法：按照数据位置排布，利用公式预测数据所在位置，再以二分法方式逼近
    $$
    Mid=low+ \frac {key-data[low]}{data[high]-data[low]}\times(high-low) \ key:要查找的键，data[high]、data[low]:剩余待查找记录最大值、最小值
    $$
    斐波拉契查找法：与二分法类似，但是以斐波拉契级数方式分割

    哈希查找法：将本身的键值通过特定数学函数运算或使用其他方法转换成相对应的数据存储地址

    哈希函数：

    1. 除留余数法：将数据除以一个常数取余数做索引
    2. 平方取中法：计算数据平方，再取中间某段数字作为索引
    3. 折叠法：将数据转换成一串数字后先拆成几个部分再相加算出桶地址
    4. 数字分析法：适用于数据不会更改且为数字类型的静态表，将重复性高的部分删除

    碰撞和溢出问题处理：

    1. 线性探测法：当发生碰撞情况，若该索引对应存储位置已有数据，则以线性的方式往后查找空的存储位置，找到位置后将数据放入
    2. 平方探测法：当溢出发生时，下一次查找地址是 (f(x)±i²) mod B ，B为4j+3型质数且1≤i≤(B-1)/2
    3. 再哈希法：当使用第一种哈希函数出现溢出就用第二种，直到没有溢出
    4. 链表法：将哈希表所有空间建立n个链表，最初默认值只有n个链表头，若发生溢出就把相同地址的键值连接在链表头后面，形成键表
