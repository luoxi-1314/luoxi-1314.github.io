利用hexo+github简单搭建个人博客

第一步：准备工作

1、下载并安装node.js

按下键盘上的 Windows + R 打开 运行窗口，输入cmd打开命令行

输入下面代码按回车查看node版本（只是为了检测是否安装了node =￣ω￣= ） 

node -v

出现下面这个界面即可

2、安装淘宝的源以提高速度

同样在cmd中输入命令安装

npm install -g cnpm --registry=https://registry.npm.taobao.org

输入这个 检测是否安装

cnpm -v

应该会出现这个界面

3、利用淘宝的源安装hex

输入下面的代码

cnpm install -g hexo-cli

安装完后输入

hexo -v

出现这个界面

恭喜你准备工作完成啦！(～￣▽￣)～

第二步：开始搭建

1、下载并安装git，官网下载很慢所以找腾讯爸爸咯！git下载

在命令行输入

git

出现一大串东西如下图表示安装完成

2、在你的电脑中找一个位置新建一个文件夹blog，这里就是以后你的博客本地所在位置啦

在你的文件夹所在位置打开命令行，如下图操作

如果不行就用git bash

在这里进行后续代码输入操作

输入代码以初始化你的博客（如果是mac或linux则在前面加入sudo用管理员身份打开）

hexo init

当提示 Start blogging with Hexo！说明创建完成，此时你的blog文件夹下会出现很多文件。

3、启动博客

命令行输入代码，开启本地运行

hexo s

在你的浏览器输入 localhost:4000 会出现默认的 Hello Word 界面

localhost:4000

那么你的本地博客已经搭建完成啦( •̀ ω •́ )✧

第三步：美化博客

1、更换主题

当然这个默认主题并不好看，我们可以在官网的主题中找一个你喜欢的主题修改就可以了官网地址,我个人使用的是Nexmoe主题，但是我推荐使用NexT主题，人气高有关教程也很多。NexT主题点击打开

下载主题并解压，将文件放在你的blog文件夹下的theme文件夹中

打开 _config文件修改theme这一行代码换成你下载的对应文件夹名称

保存后，在命令行输入三个命令（分别为清理，生成，启动）

hexo clean
hexo g
hexo s

再次输入localhost:4000就可以看见新的主题。至此完成(￣▽￣)”

第四步：远程部署

这里搭建的博客只有你自己能看得见，为了让他人看见我们将博客部署到网络上。这里使用github来部署

1、注册你的GitHub账号官网

2、新建一个仓库

输入仓库名，注意：必须是你的GitHub账户名 +.github.io

点击创建即可

3、重点来了

安装git部署插件：命令行中输入

cnpm install --save hexo-deployer-git

打开文件_config，我这里是使用sublimetext打开

像图中修改，并将打码地方输入你的GitHub账户名称

注意：每一行的冒号后面都有空格，建议使用英文输入法

输入命令即可完成部署（可能会提示你输入GitHub账号和密码）

hexo d

4、访问博客

浏览器输入你的博客地址： 账户名称.github.io

至此完成(●ˇ∀ˇ●)

其他问题

设置个人域名

觉得自己这个网页地址逼格不够，可以去购买一个域名

在解析台中添加解析

打开博客所在仓库设置

输入你的域名并保存

一段时间后打开即可

加速访问资源CDN

当我们博客中需要加入图片时，由于GitHub速度太慢，所以我们可以利用jsdelivr来加速访问

首先可以在github新建一个专门放资源的仓库，并将你的资源如图片等放入

使用方法：https://cdn.jsdelivr.net/gh/你的用户名/你的仓库名@发布的版本号/文件路径

https://raw.githubusercontent.com/luoxi-1314/mycdn/master/imp/a10.jpg – 这是github所在图片地址
https://cdn.jsdelivr.net/gh/luoxi-1314/mycdn/imp/a10.jpg –这是利用jsdelivr加速的图片地址
@版本号可以不写，默认为最新版本
